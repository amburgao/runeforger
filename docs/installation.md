# Installation

## Stable release

To install RuneForger, run this command in your terminal:

```sh
uv add runeforger
```

Or if you prefer to use `pip`:

```sh
pip install runeforger
```

## From source

The source files for RuneForger can be downloaded from the [Github repo](https://github.com/amburgao/runeforger).

You can either clone the public repository:

```sh
git clone git://github.com/amburgao/runeforger
```

Or download the [tarball](https://github.com/amburgao/runeforger/tarball/master):

```sh
curl -OJL https://github.com/amburgao/runeforger/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd runeforger
uv pip install .
```
